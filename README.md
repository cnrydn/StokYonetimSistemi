------------------------------------------------
STOK YÖNETİM SİSTEMİ

Bu proje Java Swing kullanılarak geliştirilmiş,
MySQL veritabanı ile çalışan bir stok yönetim sistemidir.

------------------------------------------------
KULLANILAN TEKNOLOJİLER

- Java (JDK 17)
- Java Swing (GUI)
- MySQL
- JDBC
- Eclipse IDE

------------------------------------------------
PROJE ÖZELLİKLERİ

- Ürün ekleme
- Ürün listeleme
- Ürün güncelleme
- Ürün silme
- Stok artırma
- Stok eksiltme
- Arama işlemi
- Kritik stok uyarısı (renklendirme)

Kritik stok seviyeleri:
- 50 ve altı: Sarı renk
- 10 ve altı: Kırmızı renk

------------------------------------------------
VERİTABANI KURULUMU

1. MySQL çalıştırılır.
2. stok_yonetim.sql dosyası çalıştırılır.
3. stock_db isimli veritabanı oluşturulur.
